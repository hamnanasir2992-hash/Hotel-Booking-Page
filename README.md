# Hotel-Booking-Page
This is a single web page I built to replicate a professional hotel booking confirmation interface. I created it from scratch using pure HTML5 and CSS3 to practice advanced layout styling and typography.

What I learned & implemented in this project:
1. Flexbox Mastery: Used Flexbox to build a structured sidebar layout and align complex item cards side-by-side.

2. CSS Box Model & Math: Used custom viewport height configurations and the calc() function to make the entire layout fit perfectly inside     the screen window without breaking or creating unwanted scrollbars.

3. Advanced Positioning: Used position: relative and position: absolute coordinate mapping to clip custom vector shapes into corners and       precisely position elements.

4. Fine-tuned UI Aesthetics: Replicated professional shadows using layered opacity structures and used border alpha transparency               modifications to create ultra-thin dividers.

Tech Stack:
1. HTML5 (Semantic structuring
2. CSS3 (Custom components, layout mechanics, and design systems)
